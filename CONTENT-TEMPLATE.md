# 교양 카드 — 최종 템플릿 (v1)

> 모든 일일 교양 항목은 이 구조를 따른다. **2026-06-24 확정.**
> 핵심 원칙: **기본 ⇄ 심화 층 구조** + **사용자 직관에 이름 붙이기** + **무거움(웹검증)**.

## 겉 (기본 — 항상 보임, 30초컷)
1. **분야 태그** (예: 경제)
2. **제목** + 영어 병기 (예: 기회비용 / Opportunity Cost)
3. **한 줄 정의**
4. **직관 고리** — 사용자가 이미 가진 직관/경험에 개념 '이름'을 붙여줌

## 속 (심화 — '더 보기'로 펼침)
5. **유래** — 누가/어떻게 만든 개념인지 (웹검증)
6. **원리** — 핵심 메커니즘, 왜 그런지
7. **예시** — 숫자 있는 것 1개 + 일상 1개 (추상 → 구체)
8. **⚠️ 흔한 착각** — 헷갈리는 개념과의 차이 / 뭐가 '아닌지'
9. **의의** — 왜 알 가치가 있나
10. **활용** — 언제·어디서·어떻게 (삶·사회생활·대화)
11. **더 배우기** — 짝 개념 · 검색어
12. **✓ 웹검증 배지**

## 보류 (나중 버전)
- "오늘 써먹기"(자기 예시 만들기) → **v2**
- 개념 연결망(여러 개념 묶기) → **v3**
- 출처 링크 직접 노출 → **앱 구현 시** (탭하면 출처)

## 완성 예시: 기회비용

**[경제] 기회비용 / Opportunity Cost**
- **한 줄:** 무언가를 선택할 때, 그 때문에 포기한 가장 나은 다른 선택지의 가치.
- **직관 고리:** (영화 예) "남은 1시간을 더 나은 데 쓸 수 있나"를 따질 때 그 '포기한 더 나은 것'이 기회비용 — 이미 쓰던 개념에 이름 붙이기.

— 더 보기 —
- **유래:** 프리드리히 폰 비저, 1914 『사회경제이론』. 비용을 '포기한 만족'으로 재정의(‘한계효용’도 그가 만듦).
- **원리:** 진짜 비용 = 명시적(돈) + 암묵적(포기한 대안). "공짜 점심은 없다."
- **예시:** ① 대학 4년 진짜 비용 = 등록금 + 못 번 월급 ② 주말 게임 5시간 = '그 시간에 할 수 있던 것'.
- **⚠️ 흔한 착각:** 이미 쓴 돈은 매몰비용(무시), 기회비용은 '안 고른 선택'의 가치.
- **의의:** 시간·기회도 비용 → 성격 다른 선택을 같은 잣대로 비교 가능.
- **활용:** "이 선택의 기회비용은?" → 후회 감소. 대화에서 "그건 기회비용이 크지" 통함.
- **더 배우기:** 매몰비용과 세트. 검색어: 기회비용 · 매몰비용 · 한계적 사고.
- **✓ 유래 웹검증** (비저, 1914)
